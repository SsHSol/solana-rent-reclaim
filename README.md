# ♻️ Solana Rent Reclaim

一键扫描并回收 Solana 钱包中所有代币账户的租金押金（Rent）。

## 功能

- 🔌 支持 Phantom / OKX / Backpack 等钱包一键连接
- 🔍 自动扫描 Token 和 Token-2022 两种程序的所有代币账户
- 📊 显示每个账户的余额、预估可回收 SOL
- 🗑️ 批量销毁余额 + 关闭账户回收租金
- 📝 支持手动输入地址只读扫描（不上链，纯查询）

## 使用

https://sshsol.github.io/solana-rent-reclaim/

## 技术栈

- 纯前端，无后端依赖
- @solana/web3.js（CDN 加载）
- GitHub Pages 部署
